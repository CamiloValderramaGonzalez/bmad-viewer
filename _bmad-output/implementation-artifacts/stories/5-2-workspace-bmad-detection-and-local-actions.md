# Story 5.2: Workspace BMAD Detection And Local Actions

## Outcome

Enable the extension to detect the current BMAD workspace, refresh data and trigger BMAD operations directly from VS Code.

## Scope

- Detect `_bmad/` and `_bmad-output/` from the active workspace
- Refresh the dashboard on workspace file changes
- Expose commands such as open viewer, refresh and sync platform integrations

## Current Status

This story is still in backlog. The current project already has BMAD detection and file watching, but not yet in a VS Code extension host.
