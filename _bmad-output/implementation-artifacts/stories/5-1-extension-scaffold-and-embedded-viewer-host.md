# Story 5.1: Extension Scaffold And Embedded Viewer Host

## Outcome

Create a VS Code extension package inside the repository that can host the BMAD viewer in a webview or view container.

## Scope

- Define extension folder structure inside this repo
- Decide whether to reuse the existing renderer directly or introduce a viewer core package
- Host the dashboard in VS Code without depending on the browser flow as the primary experience

## Current Status

This story is ready for development. We already validated that the extension approach is feasible and matches the architecture of the current viewer.
